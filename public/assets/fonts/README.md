# Font Files

Place your font files in this directory with the following naming convention:

- `Brittany.woff2` and `Brittany.woff`
- `Phatt.woff2` and `Phatt.woff`
- `Horizon.woff2` and `Horizon.woff`
- `Moontime.woff2` and `Moontime.woff`

The fonts are loaded via @font-face in `src/index.css` and are available as Tailwind utility classes:
- `.font-brittany`
- `.font-phatt`
- `.font-horizon`
- `.font-moontime`
